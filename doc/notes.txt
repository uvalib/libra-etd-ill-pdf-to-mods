Generating XML files for ingest into libra-etd

1) Export the spreadsheet enumerating the batch to a CSV file using Excel
2) Run ./scripts/convert.ksh <CSV file> to convert the format of the CSV for processing
3) Move to the pdftomods directory
4) Run ../scripts/all.ksh <batchdir> <converted CSV> to generate all the ingest XML files
5) Move to root directory
6) Run ./scripts/generateworkset.ksh <converted CSV> to generate all the ingest worksets
